About Me
I am a software engineer specializing in offensive security, including network, web, and cloud security testing, red teaming, and exploit development. I hold an M.S. in Computer Science from New York University, where I was a proud member of the Offensive Security, Incident Response, and Internet Security (OSIRIS) Lab , a student-run cybersecurity research lab. I also earned a B.A. in Physics from Columbia University, which shaped my analytical and problem-solving skills.

In addition to my professional work, I share tutorials and cybersecurity content on my YouTube channel, @exitvillain, where I cover topics like reverse engineering and web application security.

exitvillain.com

My personal cybersecurity blog that I turned into a forum for hackers. Users can post in categories, like/unlike posts, follow, customize profiles, reset passwords, send pm’s. Built with Django, JavaScript, Docker, and PostgreSQL on Amazon Cloud. Features CSRF protection, security middleware, and includes an intro video on reverse engineering.

GothamSonic.com

NYC’s largest mobile-responsive concert calendar, listing shows 3 weeks in advance. Built with React, Django REST Framework, and PostgreSQL. Deployed on Amazon Cloud using Docker, Nginx, and Gunicorn. Includes security features like environment variables and Docker secrets, with system monitoring via Prometheus.
